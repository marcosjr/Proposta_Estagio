%%%%%%%%%%%%%%%%%%%%%%%%%%%%% DESENVOLVIMENTO %%%%%%%%%%%%%%%%%%%%%%%%%

No decorrer do mês de agosto foram realizadas diversas atividades de pesquisas em artigos científicos, objetivando ampliar o conhecimento na área proposta para pesquisa. Em meio as consultas sobre obras de autores com o tema relacionado, torna-se notável a importância de se aplicar padrões de mineração de dados nas empresas ou organizações.\\
Em primeira instância foram separados vários artigos sobre mineração de dados, para leitura e entendimento do assunto, enfatizando quais são suas regras e aplicações que podem ser empregadas nas grandes bases de dados. Para que haja uma amplificação do conhecimento é necessário separa a pesquisa por módulos de processos de mineração de dados, onde cada módulo representa uma fase corrente até o objetivo final, que são os dados minerados. O maior problema do entendimento da mineração de dados está no processo de reconhecimento e descoberta de padrões. No Reconhecimento de Padrões, os resultados da aplicação de técnicas da estatística exploratória a conjuntos de padrões ou a um fenômeno esperado podem ser utilizados também para o desenvolvimento de classificadores que poderão então, subsecutivo, ser utilizados para classificar novos dados produzidos pelo mesmo fenômeno anteriormente analisado através dessas técnicas para reconhecer padrões. Para que haja sucesso nas descobertas de padrões, são utilizado alguns modelos estatísticos para gerar um conjunto de informações a partir de um conjunto inicial de dados gerado por um processo que desejamos dominar, capaz de servir para utilização em um classificador e subsequentemente ser utilizado para a classificação de novos casos gerados pelo mesmo processo que gerou os dados originais. Um outro fator importante são as técnicas exploratórias multivariadas que são projetadas especificamente para identificar padrões em conjuntos de dados multivariados ou conjuntos de dados uni variados representando sequências de observações ou comparação de um fenômeno. Uma das técnicas bastantes usadas são as técnicas baseadas em redes neurais artificiais, ou caixa preta como é definida por alguns pesquisadores. Apesar desse conceito errôneo ser tido como uma desvantagem, podemos tirar bastante aproveito em suas variadas formas de aplicação fazendo com que expanda seu espaço não se limitando somente a uma aplicação. Em fim em um processo onde se aplica a mineração de dados, pode ser adotado de vários recursos para a efetivação da atividade, dentre as técnicas citadas existem inúmeras técnicas eficazes capaz de sanar alguns problemas, pois todas os meios de mineração, reconhecimento e descobertas de padrões tem suas soluções especificas para problemas específicos.
